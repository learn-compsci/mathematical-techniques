E.g. to ask how many ways to split up 8 objects into 
            4 buckets: ^DXsPQyhV

You make 8 + (4 - 1) = 11 slots. ^KeMOSQrR

Of those 11 slots, pick any 3 of them to be bars.
Then the number of items up until the the respective bar 
represents the size of the corresponding partition. ^Ae1Tj3LE

In the above example: ^dHw9MNDi

Partition 1 has size 1 ^OBy8EtRn

Partition 2 has size 0 ^rbjw0nNS

Partition 3 has size 4 ^9uY3Fqqr

Partition 4 has size 3 ^dirQxIIr
